# PyGalaga
My version of a 1- or 2-player Galaga using Python and the library pygame.

To play, run the file `pygalaga.py`.


Controls
--------

Player 1 (Blue):
- Move: Arrow keys
- Shoot: Space bar

Player 2 (Green):
- Move: WASD
- Shoot: Left shift
